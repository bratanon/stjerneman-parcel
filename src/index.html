Open source

I'm also contributing to various open source projects including, but not limited to:

Linkit

The number one linking module for Drupal.

I created Linkit, a prominent module for Drupal, on February 19, 2010.
Linkit has garnered significant recognition over the years within the Drupal CMS community and boasts an impressive user base, with approximately 120,000+ active installations today. This module facilitates efficient linking to local resources in the CMS instance, offering features such as search autocompletion, dynamic source parsing, and a pluggable API.

For more information on Linkit, please visit https://www.drupal.org/project/linkit.

It is important to note that Linkit is freely available under the GPL2 license.
